## RSS aggregator

An application for reading and managing RSS feeds. The application checks added RSS sources every 5 seconds and adds new entries. A preview function has also been added, allowing you to view basic entry information without leaving the application.

The application is written in:
1. Vanilla JavaScript
2. The i18next library – to solve the problem of quick localization, it allows you to collect all application text and tooltips in a single file.
3. The On-change library for updating and rendering the application.
4. The Axios library for asynchronous requests.
5. The Yup library for form validation.
6. The Vite website builder
